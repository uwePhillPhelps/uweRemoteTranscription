BIT Remote Access Guide


What is it?

BIT Remote Access is an application built to initiate and manage an ssh tunnel which will allow any registered user to connect to specific BIT servers located behind the UWE Firewall.

It enables you access your Macintosh, Windows or Unix BIT homespace from any networked machine running Mac OS X.

It is a fairly simple event based program written using a combination of  applescript and shell script. It's main function is to provide a graphical frontend for ssh


Will it work on my Mac?

BIT Remote Access should work on any Macintosh (PPC or Intel) running a version of OS X later than 10.x.x


NB. It does rely on the existence of two Darwin utilities ssh and expect which are found within /usr/bin in default OS X Installations.  BIT Remote access does not at present check for their existence and it is possible that if they have been corrupted or deleted the program will not run correctly. More comprehensive environment testing is planned for future revisions.

How do I use it?


There is no installation process, simply copy BIT Remote Access to your Applications folder

(It will actually run from anywhere on your machine although we recommend the applications folder)


Double Click to Run.


Enter your BIT username and password in the fields provided ( 


Pick a server to connect to from the drop down menu

NAS for your Windows or unix files (including the public_html directory for your BIT personal website files)

Ranger for your Macintosh files


Click Connect (Enter)

When you are finished working Click Disconnect to close the tunnel


The optional Reset button will close any existing tunnels to the Remote Access Server, initialise the username and password fields and run a brief network test to establish the state of your network. 


What is Fetch?

SFTP Client from Fetchworks - we have an academic licence ...


What is ssh?

ssh or secure shell is a networking protocol that allows data to be exchanged between two networked devices using secure encryption
